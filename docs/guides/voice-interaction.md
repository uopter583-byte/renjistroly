# 语音交互

RenJistroly 提供完整的语音交互体验，包括语音输入、语音输出和 Gate 转发。

## 快速开始

### 按住说话 (Push-to-Talk)

默认快捷键 `Option + Space`：

1. **按住** `Option + Space` — 开始录音（浮动面板自动出现）
2. **说话** — 说出你的需求
3. **松开** — 自动发送并处理

> 首次使用时系统会弹出麦克风权限请求，请允许。

### 连续对话模式

点击输入框旁边的 🎤 按钮切换：

- **点按 🎤** → 进入连续监听模式
- 再次 **点按 🎤** → 停止监听并发送

在设置中可以配置：

- **发送方式**：手动（点击才发送）或自动（停顿后自动发送）
- **停顿时间**：1~4 秒可调（自动模式）

## 语音输出 (TTS)

在设置中开启「语音回复」后，AI 的文字回复会自动朗读。

- **朗读速度**：0.8x ~ 2.4x 可调
- 支持在回复过程中打断（关闭语音按钮即可）

## Gate 语音转发

Gate 是一个独立的语音中继服务，允许你：

1. 用麦克风说话
2. 语音转写为文字
3. 文字发送给 Claude Code 或其他 AI CLI
4. AI 的回复再朗读出来

### 配置

1. 进入设置 > 语音输入
2. 开启「Gate 语音转发」
3. 配置 Gate 目录路径（默认 `~/.renjistroly/gate/`）

Gate 通过文件系统交换实现进程间通信：

- `speech_in.txt` — 写入语音转写结果
- `reply_out.txt` — 读取需要朗读的回复

### 适用场景

- Claude Code 对话中解放双手
- 远程桌面/演示场景
- 无障碍辅助

## 语音状态指示

浮动面板顶部显示当前语音状态：

- 🟢 **就绪** — 等待输入
- 🔵 **聆听中** — 正在录音
- 🟠 **转写中** — 语音转文字
- 🔴 **失败** — 语音出错（点击错误栏查看详情）
