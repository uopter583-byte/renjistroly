## 产品定位

> **Codex 是 coding agent，RenJistroly 是 Mac operating agent。**

RenJistroly 不是 IDE 插件，而是能操控整个 Mac 工作环境的 AI 助手——操作应用、读取屏幕、编写代码、管理文件，同时提供企业级安全审计。
